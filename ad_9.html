Популярні товари

Перенесення макета в HTML та стилізація його в CSS.

Блок з популярними товарами в файлі index.html ми позначимо так

        <!--Популярные товары-->
            <div id="product-list">
                                        
            </div>

Пропишемо в цьому блоці заголовок другого рівня:

    <!--Популярные товары-->
        <div id="product-list">
        <h2>Популярные товары</h2>
            </div>

І створимо перший блок для товару, який буде включати в себе: найменування товару, зображення товару, короткий опис товару, ціну і кнопку:

                    <!--Популярные товары-->
                        <div id="product-list">
                            <h2>Популярные товары</h2>
                            <div class="product-item">
                                <p class="name">Наименование товара</p>
                                <div class="photo">
                                    <img src="img/photo.jpg" alt="">
                                </div>
                                <p class="desc">Описание товара</p>
                                <p class="price">30 000 руб.</p>
                                <p class="button"><button class="green">В корзину</button></p>
                            </div>
                        </div>

Кожен такий блок буде мати клас product-item. Для назви товару ми вказали абзац з класом name , для фото - блок з класом photo , для опису - абзац з класом desc , для ціни - абзац з класом price , для кнопки - абзац з класом button , так як кнопка зелена, то до тегу <button></button> ми додали клас green .

Скопіюйте блок з класом product-itemі вставте його кілька разів, щоб все вийшло шість таких блоків

У файлі зі стилями запишемо властивості для селектора #product-list :

                        #product-list {
                        margin-top: 40px;
                        overflow: hidden;
                    }

Для селектора .product-item запишемо такі властивості: колір фону, радіус заокруглення кутів, кордон, обтікання іншими елементами, відступи від блоку справа і знизу, розрахунок ширини з урахуванням меж і внутрішніх відступів, внутрішні відступи, ширину, вирівнювання тексту по центру блоку:

                        #product-list>.product-item {
                        background-color: white;
                        -webkit-border-radius: 3px;
                        -moz-border-radius: 3px;
                        border-radius: 3px;
                        border: 1px solid #ddd;
                        float: left;
                        margin: 0 20px 20px 0;
                        box-sizing: border-box;
                        -webkit-box-sizing: border-box;
                        -moz-box-sizing: border-box;
                        padding: 10px;
                        width: 226px;
                        text-align: center;
                    }

Для всіх блоків зробимо відступ знизу, а також додамо властивість overflow: hidden;:

                        #product-list>.product-item>p,
                        #product-list>.product-item>div {
                            margin-bottom: 10px;
                            overflow: hidden;
                        }

Для блоку з найменування товару вкажемо такі властивості:

                    #product-list>.product-item>.name {
                        font-weight: bold;
                        font-size: 14px;
                        line-height: 18px;
                        height: 35px;
                        color: black;
                    }

Тут ми вказали фіксовану висоту на той випадок, щоб довгі назви товарів обрізалися.

Властивості для блоку з зображенням товару:

                        #product-list>.product-item>.photo {
                        -webkit-border-radius: 3px;
                        -moz-border-radius: 3px;
                        border-radius: 3px;
                        border: 1px solid #e4e4e4;
                        width: 153px;
                        height: 137px;
                        margin: 0 auto 10px;
                    }

І властивості для самого зображення:

                    #product-list>.product-item>.photo>img {
                        width: auto;
                        height: 137px;
                    }

Властивості для блоку з описом товару:

                        #product-list>.product-item>.desc {
                        font-size: 12px;
                        line-height: 18px;
                        height: 55px;
                    }

Властивості для блоку з ціною:

#product-list>.product-item>.price {
                        color: #298f22;
                        font-weight: bold;
                        font-size: 18px;
                        line-height: 18px;
                    }

І у блоку з кнопкою прибираємо відступ знизу, а також для властивості overflow встановлюємо значення visible , щоб у кнопки була видна тінь:

                    #product-list>.product-item>.button {
                        margin-bottom: 0;
                        overflow: visible;
                    }

Блок з популярними товарами готовий. Збережіть зміни в файлах і обновіть сторінку в браузері. Переходимо до верстки посторінковою навігації.

Посторінкова навігація

Посторінкову навігацію ми створимо за допомогою ненумерованого списку:

                    <!--Пагинатор-->
                        <ul class="pagination">
                            <li class="disabled">Предыдущая</li>
                            <li class="active"><a href="">1</a></li>
                            <li><a href="">2</a></li>
                            <li><a href="">3</a></li>
                            <li><a href="">Следующая</a></li>
                        </ul>

Для відключеного пункту ми додали клас disabled, а для активного - клас acitve.

У файлі зі стилями запишемо наступні властивості:

                        .pagination {
                        clear: both;
                        -webkit-border-radius: 3px;
                        -moz-border-radius: 3px;
                        border-radius: 3px;
                        border: 1px solid #ccc;
                        overflow: hidden;
                        display: inline-block;
                        }

Для пунктів видалимо маркери, і зробимо так, щоб вони розташовувалися один за одним:

                        .pagination li {
                        float: left;
                        list-style-type: none;
                        }

Для посилання і відключеного пункту вкажемо такі властивості:

                        .pagination li a,
                        .pagination li.disabled {
                            display: block;
                            line-height: 30px;
                            vertical-align: middle;
                            background-color: white;
                            padding: 0 10px;
                            font-size: 12px;
                            text-decoration: none;
                            }

Для відключеного пункту встановимо колір тексту:

                    .pagination li.disabled {
                        color: #999;
                    }

Для активного посилання, і посилання, коли на неї вказівником пропишемо такі властивості:

                        .pagination li.active a,
                        .pagination li a:hover {
                            color: white;
                            background-color: #41b039;
                        }

Встановимо кордон між пунктами:

                        .pagination li+li a {
                        border-left: 1px solid #ccc;
                    }

Наш шаблон готовий. Тепер його потрібно протестувати.
