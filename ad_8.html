Левая колонка

Перенесення макета в HTML та стилізація його в CSS.

Перед тим, як приступити до верстки лівої колонки, змінимо внутрішні відступи у селектора #content , а також видалимо блоки з класом contentв лівій колонці і центральній частині в шаблоні index.html .

Для селектора #content вкажемо відступи зліва і справа по 29px , а також, у зв'язку з тим, що змінилася висота підвалу, відступ знизу 102px :

    #content {
        padding: 0 29px 102px;
        overflow: hidden;
        clear: both;
        margin-top: 30px;
        }

У файлі index.html після того, як ви видалите блоки з класом content з лівої колонки і центральної частини, код буде виглядати наступним чином:

    <div id="content">
        <!--Левая колонка-->
        <div id="leftcol">   

        </div>
            <!--Контент-->
            <div id="main">
            <h1>...</h1>
                 <p>...</p>
         </div>
        </div>

У файлі зі стилями для лівої колонки запишемо наступні властивості:

                    #leftcol {
                        width: 220px;
                        padding: 3px 3px 4px;
                        float: left;
                        margin-right: 18px;
                    }

Так як всередині лівої колонки будуть знаходиться блоки, які мають тінь, то ми повинні для відображення цієї тіні вказати відступи всередині padding: 3px 3px 4px; , інакше вона буде обрізатися, так як у блоку, в якому знаходиться ліва колонка, тобто у селектора #content, встановлено властивість overflow: hidden;

У файлі index.html додамо три блоки з класом block- для кошика, для пошуку за каталогом і блок для категорій товарів:

                    <div id="leftcol">
                        <!--Корзина-->
                        <div class="block">
                    
                        </div>
                        <!--Поиск по каталогу-->
                        <div class="block">
                    
                        </div>
                        <!--Категории товаров-->
                        <div class="block">
                    
                        </div>
                    </div>

У файлі зі стилями запишемо властивості для селектора .block :

                    .block {
                        -webkit-border-radius: 3px;
                        -moz-border-radius: 3px;
                        border-radius: 3px;
                        background-color: white;
                        -webkit-box-shadow: 0 1px 5px rgba(0,0,0,0.25);
                        -moz-box-shadow: 0 1px 5px rgba(0,0,0,0.25);
                        box-shadow: 0 1px 5px rgba(0,0,0,0.25);
                        margin-bottom: 20px;
                    }

Тут за допомогою властивості box-shadow ми задаємо тінь для блоку. Параметри тіні можна дізнатися у програмі Adobe Photoshop таким же чином, як ми це робили для тексту, що має тінь.

Далі ми будемо працювати над кожним блоком окремо, почнемо з кошика.

Створення блоку корзина на сайті

У цьому уроці зверстаємо блок з кошиком. У файлі index.html запишіть наступний код для даного блоку:

         <!--Корзина-->
            <div class="block">
                <div class="header">Корзина</div>
                    <div class="content">                               
                    <p>В вашей корзине:<br>нет товаров</p>                               
                    </div>
            </div>

Ми створили два блоки: для шапки <div class="header"></div>та для тематичної частини <div class="content"></div>. Тепер перейдемо до файлу зі стилями і запишемо властивості для шапки, тобто для селектора .header :

                    .block>.header {
                        padding: 10px 10px 10px 35px;
                        background-color: #41b039;
                        -webkit-border-radius: 3px 3px 0 0;
                        -moz-border-radius: 3px 3px 0 0;
                        border-radius: 3px 3px 0 0;
                        font: bold 16px/16px Arial;
                        color: white;
                    }

Тут ми вказали відступи всередині блоку, зліва зробили відступ, рівний 35px для того, щоб там вмістилася іконка. Вказали радіус заокруглення кутів, а саме верхнього лівого кута і верхнього правого кута. Вказали властивості для шрифту: накреслення, розмір, інтервал між рядками, колір.

Допишемо властивості для фонового зображення background-positionі background-repeat:

                        .block>.header {
                            padding: 10px 10px 10px 35px;
                            background-color: #41b039;
                            background-position: 10px center;
                            background-repeat: no-repeat;
                            -webkit-border-radius: 3px 3px 0 0;
                            -moz-border-radius: 3px 3px 0 0;
                            border-radius: 3px 3px 0 0;
                            font: bold 16px/16px Arial;
                            color: white;
                        }

Так як у кожного блоку в лівій колонці є своя іконка, то ми для цих блоків додамо свій клас, наприклад, для блоку з кошиком це буде клас basket: <div class="block basket"></div>:

                        .block.basket>.header {
                            background-image: url("img/basket.png");
                        }

Коли блок має два і більше класів, то ви можете записати в файлі зі стилями або один селектор класу, або все разом без пробілів,як в прикладі вище .block.basket {...}

Для блоку з класом content, що знаходиться всередині блоку з класом block , ми запишемо такі властивості:

                        .block>.content {
                            padding: 10px;
                        }

І властивості для абзацу всередині блоку:

                        .block.basket>.content p {
                            margin-bottom: 0;
                            line-height: 20px;
                        }

Переходимо до блоку з формою для пошуку.

Пошук по каталогу

Для блоку з пошуком по каталогу ми поставимо додатковий клас search :

    <!--Поиск по каталогу-->

         <div class="block search">
                                            
        </div>

Пропишемо код всередині цього блоку:

                            
                            <!--Поиск по каталогу-->
                                <div class="block search">
                                    <div class="header">Поиск по каталогу</div>
                                    <div class="content">
                                        <form>
                                            <input type="text">
                                            <button>Найти</button>
                                            <p class="description">Введите название товара</p>
                                        </form>
                                    </div>
                                </div>

Переходимо до файлу зі стилями. Спершу запишемо шлях до фонового зображення даного блоку:

                            /*Поиск по каталогу*/
                                .block.search .header {
                                    background-image: url("img/search.png");
                                }

Тепер окремо пропишемо властивості для елементів форми на початку css-документа після властивостей для абзацу:

                                    input,button,textarea {
                                    box-sizing: border-box;
                                    -webkit-box-sizing: border-box;
                                    -moz-box-sizing: border-box;
                                    display: inline-block;
                                    border-radius: 3px;
                                    -webkit-border-radius: 3px;
                                    -moz-border-radius: 3px;
                                }

Для цих елементів ми вказали властивість box-sizing , яке відповідає за те, як буде вимірюватися ширина зазначених елементів. У нашому випадку ширина блоку повинна включати в себе значення кордонів (border) і внутрішніх відступів (padding) , тому ми вказали значення border-box .

Значення inline-blockвластивості display робить елемент блоковим, але при цьому він обтекается іншими елементами веб-сторінки подібно строчному елементу.

Для текстового поля вкажемо такі властивості:

                                input[type="text"] {
                                -moz-box-shadow: inset 0 2px 3px rgba(0,0,0,0.13);
                                -webkit-box-shadow: inset 0 2px 3px rgba(0,0,0,0.13);
                                box-shadow: inset 0 2px 3px rgba(0,0,0,0.13);
                                height: 30px;
                                border: 1px solid #ccc;
                                padding: 0 4px;
                            }

Тут значення inset властивості box-shadow вказує на параметри внутрішньої тіні.

                                button {
                                background-color: #ddd;
                                background-image: -moz-linear-gradient(top, #f5f5f5, #ddd);
                                background-image: -webkit-gradient(linear, 0 0, 0 100%, from(#f5f5f5), to(#ddd));
                                background-image: -webkit-linear-gradient(top, #f5f5f5, #ddd);
                                background-image: -o-linear-gradient(top, #f5f5f5, #ddd);
                                background-image: linear-gradient(to bottom, #f5f5f5, #ddd);
                                background-repeat: repeat-x;
                                height: 25px;
                                padding: 0 10px;
                                -webkit-box-shadow: 0 3px 3px rgba(0,0,0,0.13);
                                -moz-box-shadow: 0 3px 3px rgba(0,0,0,0.13);
                                box-shadow: 0 3px 3px rgba(0,0,0,0.13);
                                border: 1px solid #c7c7c7;
                                font-size: 12px;
                                line-height: 12px;
                                cursor: pointer;
                            }

Тут фон кнопки ми вказали у вигляді градієнта, перше значення відповідає за верхній колір, друге за нижній колір. Ми вказали кілька властивостей для різних браузерів і їх версій.

За допомогою властивості cursor можна змінити вид курсора. Ми встановили значення pointer, щоб при наведенні курсору на кнопку він виглядав таким чином:

Властивості для зеленої кнопки:

                                button.green {
                                background-color: #41b039;
                                background-image: -moz-linear-gradient(top, #4ecf44, #41b039);
                                background-image: -webkit-gradient(linear, 0 0, 0 100%, from(#4ecf44), to(#41b039));
                                background-image: -webkit-linear-gradient(top, #4ecf44, #41b039);
                                background-image: -o-linear-gradient(top, #4ecf44, #41b039);
                                background-image: linear-gradient(to bottom, #4ecf44, #41b039);
                                border: 0;
                                color: white;
                                }

Запишемо стилі для текстового поля, який отримав фокус:

                            input[type="text"]:focus {
                                outline: none;
                            }

Тут ми прибрали зовнішню кордон.

Властивості кнопки, коли на неї вказівником миші:

                            button:hover {
                                background-position: 0 -25px;
                            }

Для селектора .description запишемо наступні властивості:

                            .description {
                                color: #999;
                                font-size: 12px;
                                line-height: 12px;
                            }

Переходимо до оформлення форми безпосередньо в блоці з пошуком. Для всієї форми зробимо відносне позиціонування:

                            .block.search form {
                                position: relative;
                            }

Для текстового поля встановимо ширину в 100%, щоб воно розтягувалося на все ширину блоку,в якому знаходиться. Встановимо відступ справа, щоб текст не зникав під кнопкою, і відступ знизу від поля:

                                .block.search form>input {
                                width: 100%;
                                padding-right: 57px;
                                margin-bottom: 4px;
                            }

Для кнопки встановимо абсолютне позиціонування, зробимо відступи справа і зверху, встановимо висоту, розмір тексту, міжрядковий інтервал і приберемо зовнішню тінь:

                            .block.search form>button {
                                position: absolute;
                                right: 4px;
                                top: 5px;
                                height: 20px;
                                font-size: 11px;
                                line-height: 11px;
                                box-shadow: none;
                            }

Для абзацу приберемо відступ знизу:

                            .block.search .description {
                                margin-bottom: 0;
                            }

Форма пошуку готова. Переходимо до блоку з категоріями товарів.

Bертикальне меню

Для блоку з вертикальним меню додамо додатковий клас : category :

    <!--Категории товаров-->
        <div class="block category">
                                                    
        </div>

Запишемо блок для шапки та структуру вертикального меню:

<!--Категории товаров-->
        <div class="block category">
            <div class="header">Категории товаров</div>
            <ul class="v-menu">
                <li><a href="">Благоустройство участков</a></li>
                <li><a href="">Внешняя отделка</a></li>
                <li><a href="">Внутренняя отделка</a></li>
                <li><a href="">Кровля</a></li>
                <li><a href="">Металлоконструкции</a></li>
                <li><a href="">Окна и двери</a></li>
                <li><a href="">Осветительное оборудование</a></li>
                <li><a href="">Спецтехника</a></li>
                <li><a href="">Строительный инструмент</a></li>
                <li><a href="">Цемент и бетон</a></li>
            </ul>
        </div>

Для вертикального меню ми вказали клас v-menu. У файлі style.css запишемо шлях до зображення в шапці:

                            /*Категории товаров*/
                            .block.category .header {
                                background-image: url("img/category.png");
                            }

Запишемо стилі для вертикального меню. Спершу приберемо маркери у пунктів:

                        .block.category .v-menu>li {
                            list-style-type: none;
                        }

Пропишемо властивості для посилання: зробимо її блоковим елементом, встановимо внутрішні відступи по 10px , приберемо підкреслення, встановимо кордон зверху, і вкажемо міжрядковий інтервал:

                        .block.category .v-menu>li>a {
                            display: block;
                            padding: 10px;
                            text-decoration: none;
                            border-top: 1px solid #ccc;
                            line-height: 14px;
                        }

У посилання першого пукнт приберемо кордон зверху:

                        .block.category .v-menu>li:first-child>a {
                            border-top: 0;
                        }

У посилання останнього пункту в меню вкажемо радіус округлення лівого ніженіе кута і правого нижнього кута:

                            .block.category .v-menu>li:last-child>a {
                            -webkit-border-radius: 0 0 3px 3px;
                            -moz-border-radius: 0 0 3px 3px;
                            border-radius: 0 0 3px 3px;
                            }

Встановимо властивості для посилання виділеного пункту - колір фону і тексту:

                            .block.category .v-menu>li>a:hover,
                            .block.category .v-menu>li.active>a {
                                background-color: #f8f8f8;
                                color: #444;
                            }

Вертикальне меню готове, переходимо до основної частини, а саме до блоку з популярними товарами.
