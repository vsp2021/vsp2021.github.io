Фіксований шаблон з трьома колонками

Робота з HTML CSS.

Шаблон з трьома колонками ми будемо робити на основі шаблону з двома колонками, де вузька колонка знаходиться праворуч, тому ви можете скопіювати папку fix2-right разом з її вмістом, і назвати копію fix3.

Для створення додаткової колонки нам потрібно в блоці <div id="main"></div> додати два блоки - для лівої колонки <div id="leftcol"></div>, і блок для центральної колонки <div id="centercol"></div>: 


                    <div id="main">
                        <!--Левая колонка-->
                        <div id="leftcol">
                            <div class="content"><p>Левая колонка</p></div>
                        </div>
                        <!--Центральная колонка-->
                        <div id="centercol">
                            <div class="content"><p>Центральная колонка</p></div>
                        </div>
                    </div>

У файлі зі стилями для лівої колонки запишемо ширину і властивість float з наступними значеннями:


                    #leftcol {
                        width: 180px;
                        float: left;
                    }

Для центральної колонки потрібно дописати властивість overflow із значенням hidden :


                    #centercol {
                        overflow: hidden;
                    }

Макет готовий, далі будемо створювати резиновий макет

Щоб зробити гумовий шаблон, вам потрібно змінити одиниці вимірювання ширини у селекторів #leftcolі #rightcol , а саме, замість пікселів встановити відсотки. А для селектора #wrap вказати мінімальну ширину min-width , при якій буде з'являтися горизонтальна смуга прокрутки. Наприклад, візьмемо фіксований шаблон з трьома колонками, в файлі style.css вкажемо такі властивості для селектора #wrap :


                    #wrap {
                        background-color: white;
                        min-height: 100%;
                        position: relative;
                        min-width: 700px;
                    }

І для бічних колонок замість 180px встановимо ширину, рівну 20% .

                        
                        #leftcol {
                            width: 20%;
                            float: left;
                        }
                        
                        #rightcol {
                            width: 20%;
                            float: right;
                        }
