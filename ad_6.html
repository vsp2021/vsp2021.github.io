Загальні властивості

Перенесення макета в HTML та стилізація його в CSS.

Переходимо до верстки. Візьмемо стартовий шаблон для гумового макета з двома колонками з папки \ float \ float2 даного підручника. Скопіюємо файли index.html і style.css в нашу папку на робочому столі.

У файлі index.html змінимо заголовок веб-сторінки

<title>Example</title>

І запишемо в контейнері<div id="main"></div>між тегами <div class="content"></div>заголовок і абзац:

                        <div id="main">
                          <div class="content">
                           <h1>Строительные материалы и оборудование</h1>
                            <p>Lorem ipsum dolor sit amet, consectetur adipisicing
                             elit, sed do eiusmod tempor incididunt ut labore et dolore
                              magna aliqua. Ut enim ad minim veniam, quis nostrud
                               exercitation ullamco laboris nisi ut aliquip ex ea
                                commodo consequat. Duis aute irure dolor in reprehenderit
                                 in voluptate velit esse <a href="">cillum dolore
                                  eu fugiat nulla pariaturds.</a> Excepteur sint
                                   occaecat cupidatat non proident, sunt in culpa qui
                                    officia deserunt mollit anim id est laborum.</p>
                            </div>
                        </div>

В інших блоках видаліть текст (шапка, ліва колонка, підвал).

Тепер пропишемо загальні властивості для даної сторінки в файлі style.css :

                            body {
                                background: #f3f9fa url("img/bg.jpg") repeat;
                                font: 14px/22px Arial;
                                color: #444;
                            }

background - універсальна властивість, за допомогою якого можна одночасно ставити кілька значень для фону. У нашому випадку ми вказали колір фону #f3f9fa , шлях до фонового зображення url("img/bg.jpg") , і повторення фонового зображення на сторінці repeat .

font - універсальна властивість, за допомогою якого ми можемо одночасно задати кілька значень для шрифту. Обов'язковими є розмір і сімейство шрифтів. Також ви можете вказати накреслення, стиль, міжрядковий інтервал і т.д. У нашому прикладі ми вказали розмір - 14px , міжрядковий інтервал - 22px і сімейство шрифтів - Arial .

color - колір тексту.

                            a {
                                color: #0083df;
                            }
                            
                            a:hover {
                                text-decoration: none;
                            }
                            
                            h1 {
                                font: normal 26px/26px Arial;
                                margin-bottom: 20px;
                            }
                            
                            h2 {
                                font: normal 22px/22px Arial;
                                margin-bottom: 20px;
                            }
                            
                            h1,h2,h3 {
                                color: black;
                            }
                            
                            p {
                                margin-bottom: 15px;
                            }

:hover- псевдоклас, що визначає стиль посилання при наведенні на неї курсора миші. Ми вказали властивість text-decorationі з значенням none , тобто при наведенні курсору миші на посилання у неї буде пропадати підкреслення.

За замовчуванням всі заголовки мають жирне накреслення, але в нашому макеті заголовки першого і другого рівня мають звичайне написання, тому першим значенням у властивості font ми вказали normal , потім йде розмір, міжрядковий інтервал і сімейство шрифтів.

Для всіх заголовків ми вказали чорний колір: h1,h2,h3 {color: black;}.

Видаліть властивість background-colorу селектор #wrap, #header, #footer. І властивість paddingу селектора .content.

Переходимо до верстки шапки.

Cтворення шапки

Наша шапка складається з двох блоків: блок з логотипом в лівій частині, і блок з контактами в правій частині. Запишемо в файлі index.html код для шапки:

                    <div id="header">
                        <div class="content">
                            <div class="logo">
                    
                            </div>
                            <div class="contacts">
                    
                            </div>
                        </div>
                    </div>

У блоці з класом <div class="logo"></div> пропишемо шлях до зображення за допомогою тега <img src="" alt="">. Так як логотипи в шапці зазвичай є посиланнями, що ведуть на головну сторінку, то цей тег ми укладемо між тегами <a href=""></a>:

                        <div class="logo">
                            <a href=""><img src="img/logo.png" alt=""></a>
                        </div>

У блоці з класом <div class="contacts"></div> запишемо контактну інформацію, кожен рядок помістимо в тег <p></p>:

                        <div class="contacts">
                            <p>Телефон: 8 800 2000 01 01<br>звонок бесплатный</p>
                            <p>E-mail: example@gmail.com</p>
                            <p>
                                <a href=""><img src="img/vk.png" alt=""></a>
                                <a href=""><img src="img/fb.png" alt=""></a>
                                <a href=""><img src="img/tw.png" alt=""></a>
                                <a href=""><img src="img/gp.png" alt=""></a>
                            </p>
                        </div>

Запис дзвінок безкоштовний на макеті знаходиться на новому рядку, тому перед нею ми поставили тег <br>. Іконки соціальних мереж є посиланнями, тому їх ми додали між тегами <a href=""></a>.

Переходимо до стилів. Насамперед нам потрібно вказати висоту і фонове зображення для селектора #header : 

                        #header {
                            height: 148px;
                            background: white url("img/bg-header.jpg") left top no-repeat;
                        }

Для блоку з логотипом вкажемо такі властивості: 

                        .logo {
                            margin-top: 51px;
                            float: left;
                         }

Для блоку з контактами:

                    .contacts {
                        float: right;
                        text-align: right;
                        margin-top: 25px;
                    }

Ці властивості повинні застосовуватися тільки для тих блоків, які лежать саме в шапці, тобто в блоці <div id="header"></div>, тому тут ми додамо перед стилями (.logo і .contacts) контекстні селектори :

                        #header .logo {
                            margin-top: 51px;
                            float: left;
                        }
                        
                        #header .contacts {
                            margin-top: 25px;
                            float: right;
                            text-align: right;
                        }

Для абзаців в блоці з контактами запишемо наступні властивості:

                        #header .contacts p {
                            font: bold 18px/14px Arial;
                            color: black;
                        }

Тепер нам потрібно додати додаткові класи для виділених слів: Телефон , E-mail , а такождзвінок безкоштовний . У шаблоні index.html запишіть:

                        <p><span class="green">Телефон:</span> 8 800 2000 01 01<br>
                        <span class="gray">звонок бесплатный</span></p>
                        <p><span class="green">E-mail:</span> example@gmail.com</p>

У стилях запишемо окремо для селекторів .greenі .grayкольору:

                        .green {
                            color: #1d8b15;
                        }
                        
                        .gray {
                            color: #666;
                        }

І для окремо взятих елементів в шапці запишемо розмір і накреслення шрифту:

                        #header .contacts p .green {
                            font-size: 14px;
                            font-weight: normal;
                        }
                        
                        #header .contacts p .gray {
                            font-size: 12px;
                            font-weight: normal;
                        }

З шапкою розібралися.На наступному уроці переходимо до горизонтального меню.
