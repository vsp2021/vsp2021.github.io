Фіксований шаблон з одного колонкою.

Робота з HTML CSS.

В цьому і наступних трьох уроках ми створимо шаблони, що складаються з однієї, двох і трьох колонок. Спочатку зробимо приклади для фіксованого макета.

Створіть на робочому столі окрему папку для фіксованого макета з одного колонкою, назвіть її fix1 .

Створіть у цій папці файл: index.html і папку CSS у ній файл: style.css .

У файлі index.html запишіть наступний код:

                      <!doctype html>
                        <html>
                        <head>
                            <meta charset="UTF-8">
                            <link rel="stylesheet" href="style.css">
                            <title>Фиксированный шаблон с одной колонкой</title>
                        </head>
                        <body>
                            <div id="wrap">
                        
                            </div>
                        </body>
                        </html>

Тут блок з ідентифікатором wrap буде служити контейнером для шапки, тематичної частини і підвалу. Давайте допишемо ці блоки:

                    <!doctype html>
                      <html>
                      <head>
                          <meta charset="UTF-8">
                          <link rel="stylesheet" href="style.css">
                          <title>Фиксированный шаблон с одной колонкой</title>
                      </head>
                      <body>
                          <div id="wrap">
                              <!--Шапка-->
                              <div id="header">
                      
                              </div>
                              <!--Контент-->
                              <div id="content">
                      
                              </div>
                              <!--Подвал-->
                              <div id="footer">
                      
                              </div>
                          </div>
                      </body>
                      </html>

Перед кожним блоком я додала коментар для зручності читання коду.

Тепер в кожному блоці ви можете додати який-небудь текст, щоб у вікні браузера було зрозуміло де який блок знаходиться:

                    <!doctype html>
                      <html>
                      <head>
                          <meta charset="UTF-8">
                          <link rel="stylesheet" href="style.css">
                          <title>Фиксированный шаблон с одной колонкой</title>
                      </head>
                      <body>
                          <div id="wrap">
                              <!--Шапка-->
                              <div id="header">
                                  <p>Шапка</p>
                              </div>
                              <!--Контент-->
                              <div id="content">
                                  <p>Контент</p>
                              </div>
                              <!--Подвал-->
                              <div id="footer">
                                  <p>Подвал</p>
                              </div>
                          </div>
                      </body>
                      </html>

Збережіть зміни.

Переходимо до файлу зі стилями. Спершу скинемо все відступи:

                    * {
                      margin: 0;
                      padding: 0;
                  }

Тепер запишемо властивості для блоку з ідентифікатором wrap:

                    #wrap {
                      width: 980px;
                      margin: 0 auto;
                  }

Властивість margin зі значенням 0 auto необхідно для того, щоб блок розташовувався по центру веб-сторінки. Дана властивість працює тільки в тому випадку, якщо для блоку вказана ширина.

Для шапки і підвалу вкажемо висоту:

                    #header {
                      height: 100px;
                  }
                  
                  #footer {
                      height: 50px;
                  }

Щоб виділити кожен блок кольором, давайте допишемо властивість background-color для наступних селекторів:

body {
                      background-color: #f5f5f5;
                  }
                  
                  #wrap {
                      width: 980px;
                      margin: 0 auto;
                      background-color: white;
                  }
                  
                  #header {
                      height: 100px;
                      background-color: #ddd;
                  }
                  
                  #footer {
                      height: 50px;
                      background-color: #ddd;
                  }

Збережіть зміни і відкрийте файл index.html в браузері.

Бракує відступів всередині блоків. У файлі index.html вміст блоків укладемо між тегами div з класом content . Приклад для шапки:

                      <div id="header">
                        <div class="content"><p>Шапка</p></div>
                    </div>

У файлі зі стилями для цього класу додамо відступи за допомогою властивості padding:

                      .content {
                        padding: 15px;
                    }

Як прибити підвал до низу сторінки?

Дуже часто замовники вимагають, щоб підвал був прибитий до низу сторінки. Цього можна домогтися, дописавши кілька властивостей для наступних блоків:

1. Для селекторів html і body потрібно вказати висоту, рівну100%

                        html,
                          body {
                              height: 100%;
                          }

2. Для селектора #wrap в становити мінімальну висоту min-height: 100%;, а також відносне позиціонування position: relative;:

                          #wrap {
                            width: 980px;
                            margin: 0 auto;
                            background-color: white;
                            min-height: 100%;
                            position: relative;
                        }

3.Для підвалу потрібно встановити абсолютне позиціонування position: absolute;, відступ знизу bottom: 0; , і ширину блоку width: 100%;:

                            #footer {
                            height: 50px;
                            background-color: #ddd;
                            position: absolute;
                            bottom: 0;
                            width: 100%;
                        }

Тепер підвал буде прибитий до низу сторінки. Збережіть зміни, і обновіть сторінку в браузері.

Якщо ви зменшите висоту вікна браузера, то помітите, що підвал наїжджає на вміст блоку з ідентифікатором div id="content" , щоб цього не було, додамо для останнього відступ знизу рівний висоті підвалу, тобто 50px:

                            #content {
                            padding-bottom: 50px;
                        }

збережіть лекцію у папці "лекція 1" з усіма файлами адже вони нам будуть потрібні
