Горизонтальне меню

Перенесення макета в HTML та стилізація його в CSS.

У шаблоні index.html відразу після блоку <div id="header"></div> запишемо блок для горизонтального меню з ідентифікатором navigation . У цьому блоці створимо ненумерований список для пунктів меню:

                    <div id="navigation">
                        <ul>
                            <li><a href="">Каталог</a></li>
                            <li><a href="">Блог</a></li>
                            <li><a href="">Скидки и акции</a></li>
                            <li><a href="">Прайс-лист</a></li>
                            <li><a href="">О компании</a></li>
                            <li><a href="">Контакты</a></li>
                        </ul>
                    </div>

Наше меню містить список, що випадає, тому для пункту Про компанію створимо новий список:

                    <div id="navigation">
                        <ul>
                            <li><a href="">Каталог</a></li>
                            <li><a href="">Блог</a></li>
                            <li><a href="">Скидки и акции</a></li>
                            <li><a href="">Прайс-лист</a></li>
                            <li><a href="">О компании</a>
                                <ul>
                                    <li><a href="">Для рекламодателей</a></li>
                                    <li><a href="">Для покупателей</a></li>
                                    <li><a href="">Вакансии</a></li>
                                </ul>
                            </li>
                            <li><a href="">Контакты</a></li>
                        </ul>
                    </div>

Переходимо до файлу зі стилями. Спочатку виріжемо фон для меню. Так як меню має тінь, то потрібно виділити область, захоплюючу цю тінь:

Потім для селектора #navigation пропишемо такі властивості:

                    #navigation {
                        height: 43px;
                        background: transparent url("img/bg-navigation.png") repeat-x;
                        claer: both;
                    }

Висоту беремо з урахуванням тіні. Значення transparent означає прозорий фон. Фон повинен повторюватися по горизонталі, тому ми записали значення repeat-x .

Так як в шапці для блоку <div class="contacts"></div>ми встановили властивість float: right; , то елементи наступні після цього блоку, будуть прибиті до правого краю. Щоб скасувати цю дію, для селектора #navigation ми вказали властивість clear із значенням both , тобто скасували обтікання.

Переходимо до списку. Тег <ul></ul> є прямим нащадком тега <div id="navigation"></div>. Якщо ми хочемо, щоб властивості застосовувалися тільки до дочірнього тегу певного елемента на веб-сторінці, то можемо скористатися записом для дочірніх селекторів : #navigation>ul :

              #navigation>ul {
                height: 40px;
            }

Для пунктів списку встановимо обтікання по лівому краю, кордон праворуч, відносне позиціонування і приберемо маркери:

                    #navigation>ul>li {
                        float: left;
                        border-right: 1px solid #444;
                        position: relative;
                        list-style-type: none;
                    }

Властивість display зі значенням block робить даний тег блоковим. Це властивість потрібно для того, щоб вказати відступи зверху і знизу, а також міжрядковий інтервал. Ці властивості працюють тільки для блокових елементів.

Для посилання, що знаходиться в першому пункті списку встановимо відступ зліва, рівний 29px: 

                    #navigation>ul>li>a {
                        font-weight: bold;
                        line-height: 14px;
                        color: white;
                        text-shadow: 0 1px 0 rgba(0,0,0,0.25);
                        display: block;
                        padding: 13px 28px;
                        border-right: 1px solid #333;
                        text-decoration: none;
                    }

Для активного посилання, і посилання в момент наведення на неї курсора миші встановимо такі властивості:

                    #navigation>ul>li:first-child>a {
                        padding-left: 29px;
                    }

Випадаюче меню

Список для меню, що випадає буде абсолютно позиціонується щодо пункту батьківського списку:

                    #navigation>ul>li>ul {
                        background-color: #333;
                        border-radius: 0 0 3px 3px;
                        -webkit-border-radius: 0 0 3px 3px;
                        -moz-border-radius: 0 0 3px 3px;
                        overflow: hidden;
                        position: absolute;
                        top: 40px;
                        width: 200px;
                    }

На макеті у списку нижні кути мають радіус заокруглення, рівний 3px , щоб вказати ці значення через CSS, нам буде потрібно властивість border-radius . Тут перші два значення відповідають за лівий верхній кут і правий верхній кут, другі два значення відповідають за правий нижній кут і лівий нижній кут. Префікси -webkit- і -moz- потрібні для роботи даного властивості в браузерах Safari , Chrome і Firefox відповідно.

Для пунктів списку запишемо властивість, яке приховує показ маркерів:

            #navigation>ul>li>ul>li {
                list-style-type: none;
            }

Для посилань зі списку запишемо наступні властивості:

                    #navigation>ul>li>ul>li>a {
                        font-size: 12px;
                        display: block;
                        line-height: 12px;
                        color: #eee;
                        padding: 10px 28px;
                        border-top: 1px solid #444;
                        text-decoration: none;
                    }

При наведенні курсора миші на посилання, вона буде змінювати колір фону:

                    #navigation>ul>li>ul>li>a:hover {
                        background-color: #222;
                    }

Тепер нам потрібно приховати меню, що випадає, і зробити його видимим тільки тоді, коли на пункт батьківського списку під вказівником миші, для цього допишіть властивість displayі з значенням noneдо селектору #navigation>ul>li>ul , а нижче допишіть:

                    #navigation>ul>li:hover>ul {
                        display: block;
                    }

Для селектора #contentдопишіть властивості margin-top і clear з наступними значеннями:

                    #content {
                        padding-bottom: 50px;
                        overflow: hidden;
                        clear: both;
                        margin-top: 30px;
                    }

верстка підвалу

У шаблоні index.html в блоці <div id="footer"></div> запишемо наступний код:

<div id="footer">
 
               <div class="content">
                <div class="left">
                <span>Copyright © example.com 2014</span>
                <a href="">Каталог</a>
                <a href="">Блог</a> 
                <a href="">Скидки и акции</a>
                <a href="">Прайс-лист</a>
                <a href="">О компании</a> 
                <a href="">Контакты</a>
                    </div>
                <div class="count">
                <img src="img/count.jpg" alt="">
                 </div>
                     </div>

У файлі зі стилями додамо до селектору #footer висоту, фон і кордон зверху:

                        #footer {
                        height: 72px;
                        background-color: white;
                        border-top: 3px solid #ddd;
                        position: absolute;
                        margin-bottom: 0;
                        width: 100%;
                    }

Для селектора .left запишемо властивості:

                    #footer .left {
                    float: left;
                    margin-top: 29px;
                    font-size: 12px;
                    line-height: 12px;
                    }

Для копірайтів запишемо властивості:

                #footer .left span {
                margin-right: 97px;
                }

Для посилань:

                #footer .left a {
                    text-decoration: none;
                    font-weight: bold;
                }
                
                #footer .left a+a {
                    margin-left: 17px;
                }

Для лічильника:

#footer .count {
                    float: right;
                    margin-top: 19px;
                }
