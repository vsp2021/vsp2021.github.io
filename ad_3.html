Створіть нову папку, назвіть її fix2 . Скопіюйте в цю папку файли для фіксованого макета з одного колонкою, який ми робили в минулому уроці.

Щоб створити шаблон з двома колонками нам потрібно дописати два блоки в контейнері div id="content" /div - блок для лівої колонки div id="leftcol" /div, і блок для тематичної частини div id="main" /div: .

                      <div id="content">
                        <!--Левая колонка-->
                        <div id="leftcol">
                            <div class="content"><p>Левая колонка</p></div>
                        </div>
                        <!--Контент-->
                        <div id="main">
                            <div class="content"><p>Контент</p></div>
                        </div>
                    </div>

Якщо ви збережете зміни в шаблоні і відкриєте його в браузері, то побачите, що тематична частина знаходиться під лівою колонкою. Щоб зробити її справа, нам знадобиться властивість float із значенням left. Відкрийте файл зі стилями і допишіть це властивість для селектора #leftcol , а також вкажіть для нього ширину, наприклад, 180px: 

                  #leftcol {
                    width: 180px;
                    float: left;
                    }

Збережіть зміни.

Щоб вміст лівої колонки не виходило за межі блоку div id="content" /div , додамо для цього блоку властивість overflow із значенням hidden:

                    #content {
                    padding-bottom: 50px;
                    overflow: hidden;
                    }

Теж саме потрібно зробити для блоку <div id="main"></div> :

                  #main {
                    overflow: hidden;
                  }

Шаблон з лівою колонкою готовий, переходимо до наступного етапу.

Колонка справа

Скопіюйте папку fix2 , і перейменуйте копію в fix2-right . Перейменуйте в шаблоні index.html ідентифікатор leftcolв rightcol:

        
        <div id="content">
          <!--Правая колонка-->
          <div id="rightcol">
              <div class="content"><p>Правая колонка</p></div>
          </div>
          <!--Контент-->
          <div id="main">
              <div class="content"><p>Контент</p></div>
          </div>
      </div>

У файлі зі стилями для цього блоку змініть значення у властивості float с left на right:

        #rightcol {
          width: 180px;
          float: right;
      }

Результат потрібно зберегит так як булемо використовувати шаблон в подальшому.
